# 📊 Análise de Dados em Excel | Dashboard Comercial

## 📌 Sobre o Projeto

Este projeto foi desenvolvido durante a **Semana do Excel**, promovida pela **Hashtag Programação**, com o objetivo de aplicar técnicas de análise de dados utilizando o Microsoft Excel.

Durante o projeto, foi realizado todo o processo de preparação dos dados até a construção de um dashboard interativo, seguindo uma abordagem semelhante às etapas de um projeto de Business Intelligence.

---

## 🎯 Objetivo

Transformar uma base de dados em formato **CSV** em informações úteis para análise, organizando os dados e criando visualizações que facilitem a tomada de decisão.

---

## 🛠️ Etapas Desenvolvidas

### 📥 Importação dos Dados

* Importação da base de dados em formato CSV;
* Verificação da estrutura da base.

### 🧹 Tratamento e Organização dos Dados

* Correção e organização das informações;
* Padronização dos dados;
* Ajustes necessários para garantir uma base consistente para análise.

### 📊 Análise dos Dados

* Criação de **Tabelas Dinâmicas**;
* Resumo das principais informações;
* Organização dos indicadores.

### 📈 Dashboard

Desenvolvimento de um dashboard no Excel contendo indicadores e visualizações para facilitar a análise dos dados.

---

## 🧰 Ferramentas Utilizadas

* Microsoft Excel
* CSV
* Tabelas Dinâmicas
* Dashboard em Excel
* Organização e Tratamento de Dados

---

## 📚 Competências Desenvolvidas

* Limpeza e preparação de dados;
* Organização de bases de dados;
* Construção de Tabelas Dinâmicas;
* Criação de Dashboards;
* Análise exploratória de dados;
* Visualização de informações.

---

## 🚀 Resultado

Ao final do projeto foi possível transformar uma base de dados bruta em um dashboard organizado, permitindo visualizar indicadores importantes de forma clara e objetiva.

Este projeto representa uma etapa importante no desenvolvimento das minhas habilidades em **Análise de Dados**, reforçando conhecimentos em organização, tratamento de dados e criação de dashboards utilizando Excel.
